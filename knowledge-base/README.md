## How to use this knowledge base

### For humans
- Start with `brand/identity.md` for the full picture of AI-KIP
- Use `brand/voice-and-tone.md` before creating any content
- Consult `positioning/messaging.md` for the right message in each context

### For AI agents
- Read `brand/identity.md` + `brand/voice-and-tone.md` as the foundation for every task
- Use `product/value-proposition.md` for pitch-related tasks
- Use `positioning/messaging.md` for the gold standard LinkedIn copy, pitch deck narrative, and context-specific messaging
- Reference `positioning/differentiation.md` for competitive contexts
- Check `references/compliance.md` for any claims about data privacy and hosting
- Check `product/features.md` for the complete product UI overview and feature descriptions

### Complementary sources in this repository
- `competitive-intelligence/` -- Detailed analyses of 17 competitors
- `Styleguide/AI-KIP-STYLEGUIDE.md` -- Complete design system with CSS tokens
- `carousel-template/` -- LinkedIn carousel pipeline with templates
- `carousel-template/Output final/` -- 6 gold-standard LinkedIn carousels (reference for voice & tone)
