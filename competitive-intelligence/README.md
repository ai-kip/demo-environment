# Competitive Intelligence Database

## Zweck
Systematische Analyse der Marketing-Strategien, Taktiken und Positionierung unserer Wettbewerber.
Ziel: Lernen, was Konkurrenten besonders gut machen, und anwendbare Erkenntnisse ableiten.

## Struktur

```
competitive-intelligence/
├── README.md                    # Dieses Dokument
├── competitors/                 # Ein File pro Konkurrent
│   ├── 6sense.md
│   ├── 11x.md
│   ├── 42db.md
│   ├── venta-ai.md
│   ├── luna-ai.md
│   ├── clay.md
│   └── outreach.md
├── analyses/                    # Vergleichende Analysen
│   └── channel-comparison.md    # Kanal-Vergleich aller Konkurrenten
│   └── best-practices.md        # Gesammelte Best Practices
```

## Analyse-Framework pro Konkurrent

Jeder Konkurrent wird nach folgendem Schema analysiert:

1. **Unternehmensprofil** - Kurzprofil, Gründung, Größe, Funding
2. **Positionierung & Messaging** - Value Proposition, Taglines, Kernbotschaften
3. **Zielgruppe** - ICP, Branchen, Unternehmensgrößen
4. **Marketing-Kanäle** - Welche Kanäle werden bespielt und wie
5. **Content-Strategie** - Formate, Themen, Frequenz
6. **Tone of Voice** - Kommunikationsstil und Markensprache
7. **Kampagnen & Initiativen** - Bemerkenswerte Kampagnen
8. **SEO & Organic** - Keyword-Strategie, organische Sichtbarkeit
9. **Social Media** - Plattformen, Engagement, Content-Typen
10. **Product Marketing** - Wie werden Produkte vermarktet (Demos, Trials, Tools)
11. **Was machen sie besonders gut?** - Key Takeaways & Learnings
